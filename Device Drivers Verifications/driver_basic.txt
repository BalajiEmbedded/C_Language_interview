What is Device driver:-

Device driver is a piece of code that act as a bridge between the os & hardware 
os talks software language, drivers converts the software language into hardware specific actions

Why device drivers are needed ?

OS cannot directly control hardware
Every hardware device works differently
Driver hides hardware complexity
Provides a standard interface to OS
